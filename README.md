# MCP Experiments

Personal sandbox for learning and experimenting with Model Context Protocol.

## What is MCP?
Model Context Protocol is an open standard by Anthropic that allows AI models
to connect to external tools, APIs, and data sources in a standardized way.

## Learning path
- [ ] Core concepts: servers, clients, transports
- [ ] Build first MCP server (filesystem access)
- [ ] Connect MCP server to Claude via React frontend
- [ ] Streaming UI with real-time tool call visualization
